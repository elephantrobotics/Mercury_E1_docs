# Wireless Connection Status (Connection)

In the Program interface, select the Connection function by clicking the asterisk (*), and press the C key to enter the Connection function.

<img src="../../../resources/3-FunctionsAndApplications/5-BasicApplication/5.4/5.4.7-connection/5.4.7-connection_show.png" width="300" align="center"/>

After entering the Connection function, you can select the corresponding network interface to display the configuration information of the current robotic arm network interface.

LAN1 is a static IP address.

<img src="../../../resources/3-FunctionsAndApplications/5-BasicApplication/5.4/5.4.7-connection/5.4.7-connection_LAN1.png" width="300" align="center"/>

After selection, detailed configuration information of the current network interface will be displayed.

<img src="../../../resources/3-FunctionsAndApplications/5-BasicApplication/5.4/5.4.7-connection/5.4.7-connection_LAN1_detail.png" width="300" align="center"/>

LAN1 is a static IP address.

<img src="../../../resources/3-FunctionsAndApplications/5-BasicApplication/5.4/5.4.7-connection/5.4.7-connection_LAN1.png" width="300" align="center"/>

After selection, detailed configuration information for the current network interface will be displayed.

<img src="../../../resources/3-FunctionsAndApplications/5-BasicApplication/5.4/5.4.7-connection/5.4.7-connection_LAN1_detail.png" width="300" align="center"/>

LAN2 is a static IP address. 

<img src="../../../resources/3-FunctionsAndApplications/5-BasicApplication/5.4/5.4.7-connection/5.4.7-connection_LAN2.png" width="300" align="center"/>

After selection, detailed configuration information for the current network interface will be displayed.

<img src="../../../resources/3-FunctionsAndApplications/5-BasicApplication/5.4/5.4.7-connection/5.4.7-connection_LAN2_detail.png" width="300" align="center"/>
